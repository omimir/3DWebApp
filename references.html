References

The third-party code, libraries, assets and techniques this project depends on. Anything adapted is also cited in a comment where it is used.

Libraries

  • Three.js r170. A WebGL 3D library, MIT licence. threejs.org. Vendored under js/lib/three/.
  • Bootstrap 5.3.3. A CSS and component framework, MIT licence. getbootstrap.com. Vendored under js/lib/bootstrap/.

Three.js add-on modules

These ship with Three.js under examples/jsm, MIT licence, and are vendored under js/lib/three/addons/:

  • OrbitControls for the orbit camera.
  • GLTFLoader and BufferGeometryUtils for loading the .glb models.
  • RoomEnvironment for image-based lighting.
  • EffectComposer with RenderPass, UnrealBloomPass, OutputPass, ShaderPass and supporting modules for the post-processing pipeline.
  • FXAAShader for post-processing anti-aliasing.

Assets

  • Globe surface-swap textures: earth_atmos_2048.jpg and moon_1024.jpg from the Three.js example textures (examples/textures/planets/), which derive from NASA imagery. Used for the Earth and Moon options in the texture swap.
  • Model textures: the can, bottle and globe each carry an image texture authored into the Blender source. The globe surface is a simplified Köppen climate classification map as published on Wikimedia Commons. The specific sources and licences of the model artwork are confirmed by the author in the submission.

Techniques

  • The atmosphere glow uses the fresnel rim technique that is common to Three.js planet and atmosphere examples. The GLSL in js/shaders/AtmosphereShader.js was written for this project.
  • The post-processing pipeline follows the structure of the Three.js EffectComposer examples. The vignette pass in js/postprocessing/PostFX.js uses GLSL written for this project.
